You may upload multiple spreadsheets of incoming students. Existing data will not be overwritten by subsequent uploads.

The database expects to find the following columns in the spreadsheet:

  • Id (this is the students NetId)
  • Formatted Fml Name
  • Class Year
  • Gender
  • Birthday
  • Fine Ethnic Code
  • Email
  • Blitz
  • PR Phone
  • PR Street 1
  • PR Street 2
  • PR City
  • PR State
  • PR Zip
  • PR Nation Name
  • Medical Info
  • Notes
These column names were pulled from last year's incoming student data. Let me know if the headers have changed dramatically and I can tweak the expectations of the input. The order of the columns in spreadsheet does not matter. Any extra columns will be ignored.
